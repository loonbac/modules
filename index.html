¡Paso Importante! Configurar Webhook

Para que tu módulo se actualice automáticamente, configura un webhook en tu repositorio de GitHub:

application/json

Instrucciones:

  1. Ve a tu repositorio en GitHub
  2. Settings → Webhooks → Add webhook
  3. Pega la Payload URL y el Secret
  4. Content-type: application/json
  5. Events: Just the push event
  6. Clic en Add webhook
